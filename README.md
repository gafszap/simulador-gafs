# Simulador GAFS

This repository contains a small web-based simulator for choosing **Letras Caixa** (sign letters) options and quickly requesting a quote via WhatsApp.

## Purpose

The simulator is a single HTML page that lets you pick the type of letter you want. After selecting a style, clicking **Avançar** opens WhatsApp with a prefilled message so you can request a quotation from GAFS.

## How to open the simulator

1. Clone or download this repository.
2. Open the file `simulador_gafs_final.html` in your preferred web browser. You can simply double-click the file or drag it onto an open browser window.
3. Choose a letter style and press **Avançar** to generate the WhatsApp message.
